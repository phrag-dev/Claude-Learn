# Notes Directory

Personal knowledge tracking and reference system.

## Structure

| Path | Purpose |
|------|---------|
| `LEARNING_SYSTEM.md` | Rules governing the tracker |
| `LEARNING_TRACKER.md` | Live status index of all learning items |
| `topics/` | Summary documents for CLOSED items |

## Quick Reference

- **Add item:** Tell Claude to add a topic → appended as OPEN
- **View list:** "Show learning list" → returns all non-CLOSED items
- **Update status:** "Mark X as LEARNING/CLOSED" → Claude updates tracker
- **Close item:** Requires a summary doc in `topics/` first

## PPPPP.md Hook

```
## Notes & Learning
- Rules:   Notes/LEARNING_SYSTEM.md
- Tracker: Notes/LEARNING_TRACKER.md
- Topics:  Notes/topics/
```
